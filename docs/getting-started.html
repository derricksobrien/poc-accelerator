Installation

Option 1: Local Python

cd TechConnect
python app.py

Visit: http://localhost:5000

Option 2: Docker

docker build -t poc-accelerator .
docker run -p 5000:5000 poc-accelerator

Your First POC

  1. Start the application
  2. Enter a scenario title (e.g., "AI Automation for Sales")
  3. Select solution area (AI, Security, Azure, or Cloud)
  4. Choose complexity level (L200, L300, or L400)
  5. Click "Generate"
  6. Review the generated proof-of-concept instructions

System Requirements

  • Python: 3.8 or higher
  • Memory: 4GB RAM minimum
  • Storage: 2GB free space
  • Network: Internet connection for Azure services

Environment Setup

Create a .env file in the project root:

OPENAI_API_KEY=your_api_key
AZURE_FOUNDRY_KEY=your_foundry_key
AZURE_OPENAI_API_KEY=your_openai_key
