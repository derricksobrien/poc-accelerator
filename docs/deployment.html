Deployment Options

1. Local Development

python app.py

Best for: Development and testing

Requirements: Python 3.8+, 4GB RAM

✅ No infrastructure required

2. Docker Container

docker build -t poc-accelerator .
docker run -p 5000:5000 poc-accelerator

Best for: Containerized environments

Requirements: Docker installed

✅ Portable across all platforms

3. Docker Compose

docker-compose up

Best for: Multi-service orchestration

Requirements: Docker Compose

✅ Manages all services automatically

4. Azure Container Apps

Best for: Serverless container orchestration

  • Automatic scaling based on demand
  • Built-in monitoring
  • Pay-per-execution pricing
  • No infrastructure management

✅ Fully configured and ready

5. Azure App Service

Best for: Managed web hosting

  • Managed hosting environment
  • Integrated monitoring and diagnostics
  • Easy scaling options
  • Built-in security features

✅ Fully configured and ready

Environment Variables

Configure the following environment variables for your deployment:

OPENAI_API_KEY=your_azure_openai_key
AZURE_FOUNDRY_KEY=your_ai_foundry_key
AZURE_OPENAI_API_KEY=your_openai_api_key
FLASK_ENV=production
DEBUG=false

Health Checks

The API provides health check endpoints:

GET /api/health

Returns 200 OK when healthy and ready to serve traffic.

Scaling

✅ All deployment options support horizontal scaling

  • Docker: Deploy multiple containers behind a load balancer
  • Azure Container Apps: Automatic scaling to 10+ instances
  • App Service: Built-in scale options up to 100+ instances
